⚙️ Información Técnica del Modelado

Aquí puedes poner toda la información detallada del modelo Among Us que no cupo en la tarjeta principal:

  • **Software Usado:** Tinkercad (ideal para empezar).
  • **Dimensiones Finales:** 5.4 cm x 4.4 cm x 3.0 cm (¡asumiendo un valor!).
  • **Propósito:** Práctica introductoria al Diseño 3D y uso de herramientas básicas (Tarea de Pensamiento de Ingeniería).
  • **Reto Superado:** Lograr las curvas suaves del cuerpo y el visor usando figuras básicas.

Proceso y Desarrollo

Para este modelado sencillo, se inició con un cilindro para el cuerpo principal. Luego, se utilizó una forma de caja hueca (sustracción) para crear la forma curva inferior. El visor se construyó con otro cilindro delgado. El objetivo principal era entender el flujo de trabajo: **Boceto -> Modelado CAD -> Exportación para impresión.**
